# Plan Generation Rules

## Language Requirement
- **Mandatory Chinese Output**: When generating a detailed plan, implementation plan, or long task list, the content MUST be written primarily in **Chinese**.
- This applies to:
  - `implementation_plan.md` content
  - `task.md` content
  - `walkthrough.md` content
  - Long steps/reasoning in chat responses regarding planning.

## Format
- English can be used for technical terms, file paths, and variable names where appropriate for clarity.
- The structure and explanations must be in Chinese.
